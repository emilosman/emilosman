## start session
- tmux new

## session list
- CTRL+b s

## session save + restore
- tmux resurrect plugin

## split pane
- CTRL+b % vertical split
- CTRL+b " horizontal split

## change active pane
- CTRL+b left/right/up/down arrow

## switch pane position
- CTRL+b { move current pane left
- CTRL+b } move current pane right

## close pane
- CTRL+b x

## create window
- CTRL+b c

## switch windows
- CTRL+B 0..9

## rename window
- CTRL+b ,
