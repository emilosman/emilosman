## Compile and run temp executable
go run plot.go

## Build and run executalbe
- go build -o plot plot.go
- run the compiled code: ./plot

## Run all `.go` files
`go run *.go`

## GOPATH
- Add to ~/.zshrc
```bash
export GOROOT=/usr/lib/go
export GOPATH=$HOME/go
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin
```
